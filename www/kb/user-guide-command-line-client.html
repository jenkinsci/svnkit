SVNKit command line client

SVNKit provides its own command line client realization (like svn.exe). In fact, it's a script file (specific for an OS) that invokes the main command driver class - SVN (org.tmatesoft.svn.cli), the one which redirects parameters to an appropriate command class (located in org.tmatesoft.svn.cli.command). Command classes use the SVNKit API to perform an operation.

The SVNKit command line client supports all the most important commands which svn.exe supports. Commands have got the same syntax (names, options) as svn.exe has.

Script files

In a SVNKit distribution you can find the following script files:

  • svn.bat (for Windows OS)
  • svn (shell script for Unix-like systems)

Usage

To use the SVNKit command processor (SVN class) from a command line you should have script files and library jars (such as svnkit.jar, svnkit-cli.jar) set on your PATH environment variable (script and jars must be in the same directory). Also you must have the JAVA_HOME variable set to the path where JVM is located on your system.

In addition you can use the SVN class in ant tasks, read more about how to do it.
